# Server Proxy
A Simple server proxy, just recevie request from client and forward it to remote. Work with HTTP request.

## Build
```
  git clone https://github.com/ThaiNhatMinh/Proxy-Server.git
  cd Proxy-Server
  make
 ```
## Usage
```
 ./ProxyServer <proxy port> <max thread>
```
